Regression analysis

Least squares method

The least squares method is the process of finding the best regression line for any set of data described by an equation. This method is an attempt to make the sum of the squared errors between the given objects as small as possible. The smaller the sum of squared errors, the more accurate the prediction. Linear regression allows you to predict values, so the least squares method is used by statisticians and traders who want to identify trading opportunities and trends.
